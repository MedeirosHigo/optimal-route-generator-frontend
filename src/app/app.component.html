Pour utiliser le système, suivez les étapes suivantes :

  1. Créez d'abord les marqueurs sur la carte en utilisant d'un DoubleClick. Regardez la console pour voir les données générées
  2. Cliquez sur le bouton Calculate Distance pour calculer toutes les distances entre les différents marqueurs.
  3. Visualisez les trajets sans l'optimisation de Kruskal en cliquant sur Generate Routes (without Kruskal). Cela affichera les itinéraires entre tous les points sans optimisation particulière.
  4. Appliquez l'algorithme de Kruskal en cliquant sur Calculate Optimal Distances (Kruskal) pour obtenir les itinéraires optimaux. Puis, cliquez sur Generate Routes pour visualiser les meilleures routes optimisées selon l’algorithme de Kruskal.

Remarque: Le calcul des distances optimales utilise l'algorithme de Kruskal, un algorithme glouton qui trouve un arbre couvrant minimum afin de minimiser la distance totale de déplacement.
